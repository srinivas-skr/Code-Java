# ☕ Chapter 1: Java Introduction

## 1️⃣ Definition of Java
> **"Java is high level, class based object oriented programming language which is used to build software."**

---

## 2️⃣ Programming Language vs. Language

### What is Programming Language?
> **"It is used to develop software."**
> **"It is medium to communicate b/w human and machine."**

### What is Language?
> **"It is medium to communicate b/w two entities."**

---

## 3️⃣ Software & Applications

### What is Software?
> **"Software is collection of programs that perform specific task."**
> *Ex -> windows O.S, Android O.S*

### What is Application?
> **"It is type of software helps user specific tasks."**
> *Ex -> whatsapp call, youtube watching video*

### Note:
> **"All application are S/F but not all S/F are application."**
> *i.e 1) windows O.S is S/F but not application*
> *2) Insta is both S/F & Application*

---

## 4️⃣ Types of Software

### 1) System S/F
> *1) O.S S/F (windows, linux)*
> *2) Drivers (Bluetooth)*

### 2) Application S/F
> *i) Standalone / Desktop based*
> *ii) Web application*
> *iii) Mobile application*

---

## 5️⃣ Enterprise Applications (4th Type)
> **"Enterprise applications are large scale software applications built specifically for business operations of organization."**
> *Ex -> Banking System, Hospital System etc*

> **"Mostly runs as web application but different from web application as they are used for user specific vs business specific & small-mid size vs large scale."**

---

## 6️⃣ Java Platforms
1.  **Java S.E (Standard edition):** platform for core java programs.
2.  **Java EE (Enterprise edition):** -> Jakarta EE, it is set specification / blueprint for building large scale application using Spring Boot framework.
3.  **Java FX:** -> GUI of Java's framework to build desktop application.
4.  **Android application:** built using Java with Android (API) / kotlin.

---

## 7️⃣ Java Versions
1.  **Java 8 LTS:** -> Long term support (security updates & performance).
2.  **Java 17 LTS:** -> Stable version.
